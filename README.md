# PinkGamer Discord Bot

A custom Discord bot built with Python.

## Setup

### Requirements
- Python 3.10+
- pip

### Installation

1. Clone the repo
git clone https://github.com/redgamer12886/PinkGamer-Discord-Bot.git

2. Create and activate a virtual environment
python -m venv venv
venv\Scripts\activate

3. Install dependencies
pip install -r requirements.txt

4. Create a `.env` file in the project root
DISCORD_TOKEN=your_token_here

5. Run the bot


### Functions

1. Makes a database of all the users and the money they have
2. lets you gamble
3. has basic chat response commnads
4. use !help to see a list of them all]
